## start dev server

- Start Postgres db server and modify .env file
- Create database and tables and push them to database

  ```bash
  npm run db:generate
  ```

  ```bash
  npm run db:push
  ```

- Start Server

  ```bash
  npm run db:studio
  ```

- Start Drizzle studio to manage database

  ```bash
  npm run db:studio
  ```
